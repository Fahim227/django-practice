Run Celery Beat:

python -m celery -A DjangoPracticeProject beat

Run Celery Workers:

python -m celery -A DjangoPracticeProject worker -P gevent   [gevent==22.10.2]

ToBuildContainer: docker run -publish 8080:8080 django-practice
